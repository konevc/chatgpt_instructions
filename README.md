# Useful ChatGPT instructions for Group Chats and Projects

These instructions improve the quality of LLM workflows within your Projects and Group Chats. You can adjust them to your requirements as you like.
The prompt library is updating regularly. Subscribe for the ChatGPT instructions updates.

## We are looking for contributors!

Submit your instructions to improve ChatGPT usage for all people around the world.
Read ["Contributing rules"](https://github.com/konevc/chatgpt_instructions/blob/main/CONTRIBUTING.md) for more details.

You can use our [Master Instruction](https://github.com/konevc/chatgpt_instructions/blob/main/projects/MASTER_instruction.md) to build ChatGPT project templates.

## How it works:
 1. Select, where do you like to add an instruction: "**projects**" or "**group_chats**".
 2. Find a relevant instruction, copy it and paste in ChatGPT as a project or group chat instruction.
 3. Customize instruction as you like.
 4. Type the prompt "Hi" and enjoy ChatGPT conversation!

Lists of instructions, with brief descriptions, are inside README.md in the appropriate folders.
